## Características

- ☕ **Validación automática de Java:**
  - Si tienes instalada una versión de Java que no es compatible, nosotros instalaremos la correcta por ti.
  - No necesitas tener Java instalado en tu computadora para poder ejecutar el launcher.

- 🔒 **Gestión completa de cuentas:**
  - Añade múltiples cuentas y cambia entre ellas de forma fácil y rápida.
  - Soporte completo para cuentas de Microsoft (OAuth 2.0).
  - Soporte para cuentas Offline (No-Premium / Cracked).
  - Las credenciales nunca se almacenan y se transmiten directamente a Microsoft.
